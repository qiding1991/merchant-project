模块简介:
    该模块是看看APP商家功能模块,主要是提供商家相关信息的接口服务
代码结构:
    MerchantProjectApplication
    ├── common -- 公共服务和常量
    ├── config -- 工程配置相关
    ├── model -- 公共对象封装
    ├── module -- 运维中心
    ├    ├── classify -- 商家分类模块
    ├    ├── merchant -- 商家模块
    ├    ├── param -- 接口服务输入参数
    ├── service -- 具体业务服务实现模块
    ├── utils -- 业务工具
    ├    ├── DateUtils -- 日期处理类
    └──  └── LogUtils -- 日志处理类
